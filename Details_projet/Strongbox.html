CONTEXT

The MI7 spy agency asked us to work on a new generation safe with strong authentication mechanisms. So we had several electronic cards with their own number, which could activate the authentication system, which is made up of several levels of authentication.

PROJECT IMPLEMENTATION

In this project, we had to make a truth table and several karnaugh tables representing our system.



Then we had to make logic diagrams representing our card, in my group we had card 6

Then, we made this diagram on an Arduino board (with wires and resistors) to represent our module. We then connected the Arduino board to a code allowing to realize the security system.

ACQUIRED SKILLS

During this project, we were able to learn how to make truth tables, karnaugh tables, logic diagrams.

OPINION ON THE PROJECT

I think this project has been very enriching and interesting. It was one of my first projects and it allowed me to discover the world of computing.
